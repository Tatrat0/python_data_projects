# Data Analytics Portfolio

Добро пожаловать в моё портфолио проектов по аналитике данных.

Основная цель репозитория — продемонстрировать практические навыки работы с данными на всех этапах аналитического процесса:
- подготовка и очистка данных;
- написание SQL-запросов;
- исследовательский анализ данных (EDA);
- проверка гипотез;
- построение моделей машинного обучения;
- создание интерактивных Power BI дэшбордов;
- формулирование бизнес-выводов.

## 📊 brazil_olist

Комплексный аналитический проект на основе датасета **Brazilian E-Commerce Public Dataset by Olist**.

### В проекте представлены:

- подготовка данных в PostgreSQL;
- SQL-запросы и аналитические витрины;
- исследовательский анализ данных;
- проверка бизнес-гипотез;
- Power BI дэшборды.

### Основные исследования

- влияние количества фотографий товара на количество заказов;
- анализ способов оплаты и их взаимосвязи с размером заказа.

### Используемые инструменты

`PostgreSQL` • `SQL` • `Python` • `Power BI`

Подробнее — в `brazil_olist/readme.md`

## 💼 salary

Исследование рынка труда и факторов, влияющих на уровень заработной платы.

### В проекте представлены:

- очистка и подготовка данных;
- исследовательский анализ;
- построение модели прогнозирования заработной платы;
- анализ важности признаков;
- интерактивный Power BI дэшборд.

### Основные исследования

- влияние опыта работы на уровень дохода;
- анализ профессий и профессиональных грейдов;
- исследование влияния возраста и образования на заработную плату.

### Используемые инструменты

`Python` • `CatBoost` • `SHAP` • `Power BI`

Подробнее — в `salary/readme.md`
