# Brazilian E-Commerce Analytics (Olist)

## 📌 О проекте

Аналитическое исследование открытого датасета **Brazilian E-Commerce Public Dataset by Olist**.

Цель проекта — продемонстрировать полный цикл работы аналитика данных:
- подготовка данных в PostgreSQL;
- исследование и проверка гипотез в Python;
- создание интерактивных дэшбордов в Power BI;
- формулирование бизнес-выводов и рекомендаций.

# 📊 Выполненные исследования

## 1. Влияние количества фотографий в карточке товара на количество заказов

### Гипотеза

> Увеличение количества фотографий в карточке товара не приводит к увеличению количества заказов.

### Выполненная работа

- подготовлена аналитическая витрина данных с использованием SQL;
- проведён исследовательский анализ данных (EDA);
- изучено распределение количества фотографий;
- выполнена сегментация по категориям товаров;
- снижено влияние выбросов;
- рассчитана корреляция Спирмена;
- создан интерактивный Power BI дэшборд.

### Основной вывод

Не удалось обнаружить значимую зависимость между количеством фотографий в карточке товара и количеством заказов.

---

## 2. Анализ способов оплаты

### Исследуемый вопрос

> Влияет ли способ оплаты на размер заказа?

### Выполненная работа

- исследовано распределение способов оплаты;
- рассчитана медианная сумма заказа;
- выполнена сегментация по категориям товаров;
- разработан интерактивный Power BI дэшборд.

### Основной вывод

Большинство покупок совершается с использованием кредитных карт. Для наиболее популярных способов оплаты медианная сумма заказа различается незначительно.

# 📈 Power BI

Для визуализации результатов были разработаны два интерактивных дэшборда.

## Photos vs Reviews

Позволяет исследовать взаимосвязь между количеством фотографий в карточке товара, количеством заказов и пользовательскими оценками.

Фильтрация:
- категория товара;
- цена;
- количество фотографий;
- пользовательский рейтинг.

---

## Payment Analytics

Позволяет анализировать распределение способов оплаты и их связь с медианной суммой заказа.

Фильтрация:
- тип оплаты;
- категория товара.
